Installation
============

1. Install leiningen -- https://github.com/technomancy/leiningen
2. clone Calaja project
3. run lein deps, compile, run

Controls
========

Player One
----------
Rotate:	left and right arrow keys
Thrust:	up arrow key
Shoot:	down arrow key

Player Two
----------
Rotate:	A and D
Thrust:	W
Shoot:	S
